Démo de quelques fonctionnalités d'Electron

IPC: Renderer vers Main (Unidirectionnel)

Dans cette communication, renderer envoie un message à main mais ne reçoit pas de réponse (unidirectionnel).

  • Renderer (exposé via preload script) : ipcRenderer.send(channel, data)
  • Main : ipcMain.on(channel, (event, data) => {...})

Documentation : Pattern 1 Renderer to main (one-way)

IPC: Renderer vers Main (Bidirectionnel)

Dans cette communication asynchrone, renderer appelle main et reçoit une réponse.

  • Renderer (exposé via preload script): ipcRenderer.invoke(channel, data)
  • Main: ipcMain.handle(channel, callback)
